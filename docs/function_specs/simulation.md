# 3. FIRE 시뮬레이션 세부 명세서

### 3.1. 시점에 따른 구분
1. 사용자는 최대 3개의 시뮬레이션 케이스를 생성하고 이름을 설정할 수 있다.

### 3.2. 시뮬레이션 입력 변수 (Inputs)
#### 3.2.1. 사용자 정보 (User Background)
1. **연령 계산**: [계정 설정](./account.md)에 입력된 생년월일 정보를 기반으로 현재 연령 및 시뮬레이션상의 미래 연령을 자동으로 산출한다.
2. **시뮬레이션 기간 (Duration)**: 시뮬레이션을 몇 년 뒤까지 수행할지 설정한다.
    - **기본값**: 사용자가 한국 나이 기준(혹은 만 나이) **만 90세**가 되는 해까지를 기본 시뮬레이션 종료 시점으로 설정한다.
    - **수정 가능**: 사용자는 필요에 따라 시뮬레이션 종료 연령(기간)을 직접 수정할 수 있다.

#### 3.2.2. 자산 (Assets)
1. **기초 순자산**: 포트폴리오에서 포함할 항목(자산)을 선택하거나 직접 금액을 입력한다.

#### 3.2.3. 수입 전략 (Income Strategy)
미래에 발생할 예상 수입을 설정하여 자산 시뮬레이션에 반영한다.
1. **수입 금액 및 카테고리 설정**: 사용자가 직접 수입 금액과 해당 수입원의 카테고리를 입력할 수 있다. (예: 직장 근로소득, 부업, 아르바이트, 연금 등)
2. **기간별 수입 산정**: 사용자가 직접 설정한 연령 구간(기간)에 따라 다른 금액과 카테고리를 개별 설정할 수 있다. 사용자는 시작 연령과 종료 연령을 자유롭게 지정하여 수입 구간을 나눈다.
    - 예시 1: 현 연령 ~ 40세(사용자 지정)에는 월 300만원 직장 근로소득 + 월 200만원 부업
    - 예시 2: 40세 ~ 65세(사용자 지정)에는 월 100만원 아르바이트
    - 예시 3: 65세 ~ 시뮬레이션 종료 시점에는 월 100만원 연금 + 월 50만원 아르바이트
3. **수입 상승률**: 설정된 수입에 대해 매년 연 복리로 적용될 수입 상승률(예: 연봉 인상률, 물가 연동 등)을 사용자가 직접 설정하여 반영할 수 있다.

#### 3.2.4. 지출 및 인출 전략 (Spending & Withdrawal)
사용자는 다음 중 하나의 인출 방식을 선택한다:
1. **고정 금액 방식**: 매달/매년 정해진 금액을 인출하여 생활비로 사용한다. 
    - 가계부 데이터 연동: 가계부에 기록된 "최근 평균 지출액" 또는 "최소 생계비(필수 카테고리 합계)"를 참고용 기본값으로 자동 제안한다.
2. **자산 비율 방식**: 매년 초 전체 자산의 일정 비율(%)을 인출하여 사용한다. (예: 4% 룰)
3. **추가 예비비**: 특정 시점(예: n년 뒤) 혹은 주기적으로 발생하는 비정기 대 지출(교육비, 결혼 등)을 별도로 추가/관리할 수 있다.

#### 3.2.5. 경제 가정 (Economic Assumptions)
1. **물가상승률**: 미래 지출액 계산에 반영될 예상 물가상승률을 설정한다.
2. **연 수익률**: 보유 자산 전체에 대한 연간 기대 수익률을 설정한다.

### 3.3. 시뮬레이션 결과 (Outputs)
1. **자산 추이 그래프**: 연도별 자산 잔액의 변화를 선 그래프로 시각화한다.
2. **은퇴 리포트 (Safe Zone 분석)**: 
    - **안정 구역 (Safe Zone)**: 임의의 시점(은퇴일) 이후부터 사용자가 설정한 **시뮬레이션 종료 시점(기본 만 90세)**까지 시뮬레이션상의 자산 잔액이 지속적으로 0보다 큰 상태를 유지하는 구간. 
        - `Asset(t) > 0` (for all `t` from `Retirement Date` to `Simulation End Age`)
    - **영구 안정 구역 (Perpetual Safety)**: 연간 자산 기대 수익이 물가 상승률을 반영한 연간 인출액보다 크거나 같아, 원금이 보존되거나 늘어나며 자산이 영구적으로 고갈되지 않는 상태.
        - `자산 총액 * 기대수익률 >= 연간 인출액(물가상승률 반영)`
    - **결과 출력**:
        - 자산이 소진되지 않는 '안정 구역' 진입 시점(은퇴 가능일) 도출
        - 영구 안정 달성 여부 및 시점 표시
        - 자산 고갈 예상 시점 및 경고 메시지 출력
