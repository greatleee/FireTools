# 4. 계정 및 시스템 세부 명세서

### 4.1. 사용자 인증 및 권한 (Auth)
1. **회원가입 및 로그인**:
    - 이메일/비밀번호 기반 회원가입 및 로그인
    - 소셜 로그인 연동 (Google, Apple 등 OAuth 지원)
2. **비밀번호 관리**:
    - 비밀번호 찾기(이메일 인증 등) 및 재설정
    - 비밀번호 변경 기능

### 4.2. 보안 및 세션 관리
1. **세션 유지 및 자동 로그인**:
    - JWT 등 토큰 기반 인증 프로세스
    - 자동 로그인 설정 및 토큰 만료 시 갱신(Refresh) 처리

### 4.3. 사용자 프로필 및 환경 설정
1. **프로필 관리**:
    - 사용자 닉네임 편집
    - 생년월일 입력: 미래 자산 흐름 시뮬레이션 시 연령 계산 등에 활용
    - 선호 통화(단위) 설정 (KRW, USD 등 기본 환율 계산 시 노출 단위)
