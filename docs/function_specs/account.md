# 4. 계정 및 시스템 세부 명세서

### 4.1. 사용자 인증 및 권한 (Auth)
1. **회원가입 및 로그인**:
    - 이메일/비밀번호 기반 회원가입 및 로그인
2. **비밀번호 관리**:
    - 비밀번호 찾기(이메일 인증 등) 및 재설정
    - 비밀번호 변경 기능

### 4.2. 보안 및 세션 관리
1. **세션 유지 및 자동 로그인**:
    - JWT 등 토큰 기반 인증 프로세스
    - 자동 로그인 설정 및 토큰 만료 시 갱신(Refresh) 처리

### 4.3. 사용자 프로필 및 환경 설정
1. **프로필 관리**:
    - 사용자 닉네임 편집
    - 생년월일 입력: 미래 자산 흐름 시뮬레이션 시 연령 계산 등에 활용
    - 선호 통화(단위) 설정 (KRW, USD 등 기본 환율 계산 시 노출 단위)

### 4.4. 계정 초대 및 공유 (Family Sharing)
배우자 혹은 가족과 데이터를 함께 관리하기 위한 기능을 제공한다.
1. **계정 초대**:
    - 이메일 또는 초대 링크를 통해 다른 사용자를 초대할 수 있다.
    - 초대받은 사용자가 수락하면 동일한 가계부 및 포트폴리오 데이터를 공유한다.
2. **데이터 동기화**:
    - 여러 명의 사용자가 동시에 데이터를 입력/수정할 경우 실시간으로 동기화되어 시뮬레이션에 반영된다.
