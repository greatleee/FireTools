# 1. 가계부 (지출 트래킹) 세부 명세서

### 1.1. 지출 입력 방식 (2-Track)
사용자의 편의에 따라 두 가지 입력 방식을 모두 지원한다.
1. **간편 입력**: 원하는 월(month)의 총 지출액만 입력. 바로 시뮬레이션을 돌려보고 싶은 경우 주로 사용한다.
2. **상세 입력**: 날짜, 금액, 카테고리, 메모를 포함하여 건건이 상세 내역을 기록한다.

### 1.2. FIRE 맞춤형 카테고리 (분류)
모든 지출을 생존과 직결된 "필수 지출"과 통제 가능한 "선택 지출"로 구분하여 기록한다.
1. **필수(Needs) / 선택(Wants) 태그**: 모든 지출 내역에 필수 또는 선택 속성을 부여하여 최소 생계비를 쉽게 파악할 수 있도록 한다. (예: 주거비-필수, 외식비-선택)
2. **커스텀 카테고리**: 사용자가 자신의 라이프스타일에 맞게 카테고리를 자유롭게 추가/수정/삭제할 수 있다.

### 1.3. 예산 및 지출 통제 (Budgeting)
1. **월별 목표 지출액(예산) 설정**: 이번 달 목표 지출액을 설정하고, 현재까지의 실시간 투입 달성률을 시각화(진행률 바 등)하여 지출 페이스 조절을 돕는다.
2. **무지출 데이(Zero-Spend Day) 트래킹**: 달력 형태 UI에서 지출이 0원인 날을 시각적으로 강조(배지 혹은 스탬프)하여 절약 성취감을 유도한다.

### 1.4. 분석 및 FIRE 시뮬레이션 연동
1. **지출 트렌드 분석**: 최근 N개월간의 월별 지출 추이를 막대그래프로, 카테고리별 지출 비율을 원형 그래프로 제공한다.
2. **시뮬레이션 기본값 연동**: 가계부에 기록된 데이터를 바탕으로 "최근 평균 지출액" 또는 "최소 생계비(필수 카테고리 합계)"를 자동으로 추출하여 FIRE 시뮬레이션의 기본값으로 자동 제안한다.
