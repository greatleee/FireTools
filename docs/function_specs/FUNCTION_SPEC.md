# FireTools

FIRE(Financial Independence, Retire Early)를 위한 도구 모음 애플리케이션

## 1. 가계부 (지출 트래킹)
FIRE 시뮬레이션과 연동하기 위한 지출 전용 기록 및 통제 도구입니다.

- **기능 카테고리:**
  - 지출 입력 방식 (2-Track)
  - FIRE 맞춤형 카테고리 (분류)
  - 예산 및 지출 통제 (Budgeting)
  - 분석 및 FIRE 시뮬레이션 연동
- [상세 세부 명세서 보기](./expense.md)

## 2. 포트폴리오 관리
사용자는 자산을 논리적인 그룹인 '포트폴리오'로 묶어 관리하며, 포트폴리오 하위에 실제 '계좌'를 연결합니다. (구조: 포트폴리오 > 계좌 > 자산)

- **기능 카테고리:**
  - 포트폴리오 및 계좌 관리
  - 자산 관리 (계좌 내 편입)
  - 포트폴리오 성과 분석
- [상세 세부 명세서 보기](./portfolio.md)

## 3. FIRE 시뮬레이션
가계부의 지출 관리와 포트폴리오 관리 데이터를 기반으로 미래 자산 흐름을 시뮬레이션합니다.

- **기능 카테고리:**
  - 시점에 따른 구분
  - 시뮬레이션 입력 변수 (Inputs)
  - 시뮬레이션 결과 (Outputs)
- [상세 세부 명세서 보기](./simulation.md)

## 4. 계정 및 시스템 (사용자 프로필 및 인증)
앱 사용을 위한 기본적인 계정 관리, 인증 및 프로필 기능을 제공합니다.

- **기능 카테고리:**
  - 사용자 인증 및 권한 (Auth)
  - 보안 및 세션 관리
  - 사용자 프로필 및 환경 설정
- [상세 세부 명세서 보기](./account.md)
